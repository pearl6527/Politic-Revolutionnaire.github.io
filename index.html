How to contribute
Go to github.com then register an account and sign in. Next go to desktop.github.com and download Github Desktop, login and close it. Then go to https://github.com/Politic-Revolutionnaire/Politic-Revolutionnaire.github.io and click fork repository. Now go to the GitHub application now on your computer and chose a directory to save your files. Go to that directory once you are finished and go to Politic-Revolutionnaire.github.io and add to the notes as you please. Once finished return to the GitHub application on your desktop and select sync and create pull request. The pull request will merge your changes with the updates to the website once approved. Congrats you have become a contributor to Politic-Revolutionnaire.github.io
